<h3 align="center">This is a node script to scrap car.gr dealers by region and export them in excel format.</h3>


<h4>Install & Use</h4>

```
1. git clone https://github.com/vpsnak/cargr-dealer-scrapper.git
2. cd cargr-dealer-scrapper
3. npm install
4. node ./index.js
```

You can tweak vehicles you want to export in config.js (vehicleTypesToExport). Data stores in db.json by default and should be cleared after each export to prevent dublicate entries.
